Поисковый движок (Search Engine)
Дипломный проект курса "Java-разработчик"

О проекте
Backend-часть поисковой системы, аналогичной по принципу работы Яндекс Серверу. 
Приложение самостоятельно обходит сайты, 
индексирует их страницы и выполняет поиск по проиндексированным данным.

Технологический стек
Java 21

Spring Boot 3.3.4 (MVC, Data JPA)

MySQL, Hibernate ORM

Maven, Git

Лемматизатор: Russian Morphology for Lucene

Jsoup (парсинг HTML)

ForkJoinPool (многопоточность)

Функциональность
Многопоточная индексация сайтов (обход всех страниц)
Лемматизация текста и подсчет частотности слов
Индексация новых страниц по URL
Поиск по проиндексированным страницам с ранжированием по релевантности
Статистика по сайтам, страницам и леммам
REST API и веб-интерфейс для управления

Цель проекта
создать полноценный поисковый движок, 
демонстрирующий навыки backend-разработки: 
работа с многопоточностью, базами данных, 
парсингом веб-страниц и морфологией русского языка.
